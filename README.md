# Business Sales Performance Dashboard

This project is a Power BI dashboard designed to analyze and monitor overall business sales performance.

## Overview
The dashboard provides insights into sales, profit, quantity sold, returns, customers, products, regions, and time-based trends. It helps identify patterns, performance gaps, and growth opportunities across different business dimensions.

## Key Features
- Sales, profit, and quantity KPIs  
- Category-wise and product-wise performance  
- Customer behavior analysis  
- Region-wise gross profit comparison  
- Time-based sales trends  
- Return analysis for performance evaluation  

## Tools Used
- Power BI  
- Business Sales Dataset
